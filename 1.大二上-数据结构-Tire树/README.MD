
#Trie树（字典树）#

实现Trie树结构，我主要使用了多重链表来实现。（C++/MFC）

还基于已有Trie树实现其基本应用，如词频统计，智能提示（搜索建议），字符串字典顺序排序。

程序可以根据输入数据，动态构造Trie树并展示，其中一些基于Trie树的应用程序也**可以动态展示节点访问过程**。
